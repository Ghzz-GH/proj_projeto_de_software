# 🛒 FeiraFlix

Plataforma de assinaturas semanais para entrega de produtos frescos de feira.

---

## 📌 Sobre o Projeto

O **FeiraFlix** é uma solução desenvolvida na disciplina de Projeto de Software com o objetivo de facilitar o acesso a alimentos frescos por meio de um sistema de assinatura.

A proposta conecta consumidores a produtores locais, incentivando a economia circular e o consumo consciente.

---

## 🎯 Problema

Muitas pessoas têm dificuldade em:
- Acessar produtos frescos regularmente
- Encontrar feiras próximas
- Manter uma alimentação saudável na rotina corrida

---

## 💡 Solução

O FeiraFlix resolve isso oferecendo:
- Assinatura semanal de cestas
- Entregas programadas
- Interface simples para gestão dos pedidos
- Incentivo ao consumo sustentável
